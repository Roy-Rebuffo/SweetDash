# confectioneryProject
🎯 Objetivo del Proyecto Desarrollar una aplicación de gestión para una tienda de repostería, que facilite el control de pedidos, productos, ingredientes, stock y clientes, ayudando a automatizar las tareas manuales actuales de la repostería. En el futuro, esta aplicación se ampliará para incluir una base de datos y una interfaz gráfica de usuario.
